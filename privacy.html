Privacy Policy for Lens Last updated: 2026-01-20 Lens is a Chrome extension designed to provide brief perspective and reflection prompts to support clearer thinking. Data Collection Lens does not collect, transmit, or store any personal user data on external servers. All content and functionality are delivered locally within the extension. Local Storage Lens uses local browser storage to: • Remember recently shown prompts • Store basic user preferences (such as settings) This data remains on the user’s device and is not shared or transmitted. User Input Any optional text entered by the user is processed locally and is not sent, stored remotely, or analyzed by external services. Third-Party Services Lens does not use third-party analytics, tracking tools, or advertising services. Remote Code All executable code is bundled with the extension at install time. Lens does not load or execute remote code. Changes to This Policy If Lens’s functionality changes in a way that affects data handling, this privacy policy will be updated accordingly.
